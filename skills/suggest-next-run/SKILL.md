# Suggest Next Run

## Instructions

1. Use insights from previous experiments.

2. Identify:
   - best-performing parameter ranges
   - unexplored or underexplored areas

3. Recommend:
   - next experiment configuration
   - parameter values to try
   - expected outcome

4. Provide:
   - reasoning for recommendation
   - confidence level (low, medium, high)

5. Ensure recommendation is:
   - realistic
   - based on observed trends
