# 🧠 IteraMind

> Learn from every experiment. Improve every next run.

---

## 🚀 Overview

IteraMind is an AI agent that transforms raw experiment data into actionable intelligence.

Instead of relying on trial-and-error, IteraMind:
- analyzes past experiments
- detects performance patterns
- identifies what works (and what doesn’t)
- recommends the next best experiment configuration

---

## ❗ Problem

Machine learning workflows are often inefficient:
- experiments are run randomly
- insights are not systematically captured
- teams repeat mistakes

---

## 💡 Solution

IteraMind introduces **experiment intelligence**:
- structured analysis of past runs
- pattern detection across parameters
- data-driven recommendations for next experiments

---

## 🧩 Agent Design (gitagent format)

### Core Files
- `agent.yaml` → agent definition
- `SOUL.md` → identity & behavior
- `RULES.md` → constraints
- `skills/` → modular capabilities

### Skills
- `analyze-experiments` → extracts metrics & comparisons
- `detect-patterns` → finds correlations & anomalies
- `suggest-next-run` → recommends next experiment
- `summarize-insights` → outputs actionable learnings
